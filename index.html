Helper

This visualization supports standard interactions, including panning, zooming, node manipulation, as well as filtering through URL query filters. Multiple filters can be combined.

The graph is constructed from the first 15 minutes of the first 2020 U.S. presidential debate between Donald Trump and Joe Biden (transcript available at:[link], video available at:[link])

Basic interactions

  • Drag empty space to pan the graph.
  • Scrolling (using mouse wheel or trackpad) controls zoom level.
  • Nodes can be repositioned via drag-and-drop.
  • Double-click (or double-tap) a node to open its associated text.

Query parameters

The interface support the following query parameters:

Query What it does
showClasses=1|0 Show or hide CLASS nodes and related links. Default: 1.
showTextOf=1|0 Show or hide TEXTOF nodes. Default: 0.
AG=1|0 (or ag) Argument-only graph mode (only ARGUMENT↔ARGUMENT links). Default: 0.
debaters=... Filter by debater(s). Supports aliases (e.g. Trump, Biden), comma lists, JSON arrays, or repeated params.
ac=... (or AC) Filter by argument index/range (display index). Examples: ac=5, ac=5-12, ac=1,3,7-9.
acceptance=... Filter by reasoning score. acceptance=0.5 means score ≥ 0.50; acceptance=0.50-0.60 means inclusive range.

Examples

?acceptance=0.5

?acceptance=0.50-0.60&AG=1

?debaters=Trump&ac=1-12&showTextOf=1

?debaters=["Biden","Trump"]&ac=5,8-10&acceptance=0.45-0.80
